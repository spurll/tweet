# tweet

Tweets.

# Usage

```sh
$ ./tweet.py "Look, it's a tweet!"
```

## Setup

You'll need `tweepy`, which is available on PyPI. You'll also need to [create a Twitter app](apps.twitter.com/app/new)
and supply its API keys, which is pretty straightforward.
